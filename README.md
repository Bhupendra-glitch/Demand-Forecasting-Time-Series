# 📊 Demand Forecasting using Time Series Models

## 📌 Project Overview
Developed time-series forecasting models to predict future sales demand using historical sales data.

## 📊 Dataset
- 3+ years of historical sales data
- 1,000+ time points

## 🎯 Objectives
- Detect seasonality and trend patterns
- Compare forecasting models
- Improve forecast accuracy

## 🔍 Models Implemented
- ARIMA
- Exponential Smoothing
- Holt’s Linear Trend
- ML-based Forecasting Models

## 📊 Performance
- Reduced forecast error (RMSE/MAPE) by 15%+
- Outperformed naïve baseline model

## 🛠️ Tech Stack
- Python
- Pandas
- Statsmodels
- Scikit-learn
- Matplotlib

## 📈 Key Insights
- Identified weekly and seasonal sales patterns
- Improved inventory planning decisions
